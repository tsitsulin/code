### Operators and spaces

1. Operators like `If`, `foreach` etc. are separated by newline(s).

```php
function (): void
{
    $var1 = 100;
    
    if (!is_int($var1)) {
        // Do something
    }

    if ($var1 > 100) {
        // Something do ...
    }
    
    $array1 = [1, 2];
    
    foreach ($array1 as $item) {
        // Do some logic
    }
}
```
